# gv_menucard
Ein einfaches anpassbares Speisekarten Skript für FiveM.

# Funktionen
- Öffnen einer Speisekarte über ein Item
- Animation während des Lesens (Clipboard in der Hand)
- Doppelseitige Speisekarte mit umblättern Funktion
- Standalone unabhängig von anderen Systemen

# Installation
1. Lade dir dieses Dateiensammlung runter und füge sie in deinen ressource Ordner ein.
2. Fügen Sie diese Zeile in ihre server.cfg ein.
   ```plaintext
   ensure gv_menucard
   ```
3. Starten Sie ihren Server neu.

# Konfiguration
Tauschen Sie die Bilder unter dem Pfad:
  ````plaintext
  gv_menucard/html/img/*.png
  ````
gegen ihre eigenen aus. Beachten sie dabei folgende Dinge:
- Format .jpg
- maximal 5 Bilder
- Name: {Seite1.jpg, Seite2.jpg, Seite3.jpg, Seite4.jpg, Seite5.jpg}
